# thesession.org_member_sets_incipits
thesession.org member session sets incipits

Creates a self contained HTML of a members' sets in incipits form, so it can be used offline. 
The intent is to be usable as a prompt at a music session to find your known tunes/sets
If you do not like technology at the table during sessions don't use it. 

Searchable/Filterable by 
- set name
- key
- tune type
- tag

Ordered by set date/name ASC/DESC

Set name links back to thesession.org
